## Repository Structure

Solutions are auto-organized by NeetCode's GitHub Sync into topic folders, then by problem, with each submission saved as a separate file:

```
Data Structures & Algorithms/
  <problem-id>/
    submission-0.java
    submission-1.java
```

## How This Repo Stays Updated

Every accepted submission on [NeetCode.io](https://neetcode.io) is auto-committed here via their [GitHub Sync](https://neetcode.io/profile/github) feature — no manual copy-paste, which keeps the commit history an honest reflection of actual practice frequency rather than batched dumps.
